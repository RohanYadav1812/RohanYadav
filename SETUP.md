# GitHub Profile Setup

Your profile README will appear at **https://github.com/RohanYadav1812** when you push this.

## Steps

1. **Create the repo** (if not exists):
   - Go to https://github.com/new
   - Repository name: **RohanYadav1812** (exactly your username)
   - Public, no README needed
   - Create

2. **Push this folder**:
   ```powershell
   cd d:\git\github-profile
   git init
   git add .
   git commit -m "Add profile README"
   git branch -M main
   git remote add origin https://github.com/RohanYadav1812/RohanYadav1812.git
   git push -u origin main
   ```

3. **Update LinkedIn** in README.md with your actual LinkedIn URL if different.

4. **Customize** - Edit README.md to add more projects, change colors, or add new sections.
